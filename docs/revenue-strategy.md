## 2. SEO以外の集客チャネル戦略

### 2-1. X(Twitter) — 最優先チャネル

**なぜ最優先か**:
- 即日開始可能、費用0円
- FXトレーダーはXの利用率が極めて高い
- SEOが立ち上がるまでの「つなぎ」として唯一の生命線
- バズ1回で数千PVが発生する可能性がある

**アカウント設計**:
- アカウント名: 「FXコンパス@初心者のための地図」
- プロフィール: 「国内FX口座の比較/スプレッド/資金管理を出典付きで整理。海外FXは扱いません。サイト→hajimete-fx.com」
- 固定ツイート: 最もCVに近い記事（kouza-ranking.html）へのリンク

**投稿カテゴリと頻度**:

| カテゴリ | 頻度 | 内容 | 目的 |
|---------|------|------|------|
| 相場コメント | 毎日1-2回 | 当日のドル円/ユーロ円の動き+初心者向け解説 | エンゲージメント獲得 |

### 2-3. プログラマティックSEO — 大量ページ自動生成

**考え方**:
同じテンプレートで変数（通貨ペア、FX会社名、条件）を差し替えて大量のページを生成し、ロングテールキーワードを網羅的に拾う手法。Wiseは通貨コンバーターで850万ページを生成し、圧倒的なSEO流入を獲得した実績がある。

**FXコンパスで実行可能なパターン**:

#### パターンA: 通貨ペア別ページ
- テンプレート: 「[通貨ペアA/通貨ペアB] の特徴・スプレッド比較・おすすめ口座」
- 対象: 主要20通貨ペア x 解説ページ = 20ページ
- キーワード例: 「ポンド円 スプレッド 比較」「豪ドル円 スワップ」
- 差し替えデータ: スプレッド値、スワップ値、ボラティリティ、特徴

#### パターンB: FX用語 x 通貨ペア
- テンプレート: 「[用語]とは？[通貨ペア]での具体例」
- 対象: 30用語 x 5通貨ペア = 150ページ
- キーワード例: 「スプレッド ポンド円」「レバレッジ ドル円 いくらから」

#### パターンC: VPS x 用途
- テンプレート: 「[VPSサービス] で [EA名/MT4/MT5] を動かす手順」
- 対象: 4サービス x 10パターン = 40ページ
- キーワード例: 「ConoHa VPS MT4 設定」「XServer VPS EA 自動売買」

**実装方法**:
静的HTMLなのでビルドツールなし。Node.jsスクリプトでテンプレートHTMLにJSONデータを流し込み、htmlファイルを出力する。Cloudflare Pagesにデプロイ。

**生成ページ数**: 210ページ（A:20 + B:150 + C:40）
**見込みPV**: 1ページあたり月10-50PV x 210 = 月2,100-10,500PV

**工数**: テンプレート設計3日 + データ整備2日 + 生成スクリプト1日 = 初期6日。以後はデータ更新のみ。

**注意**: Googleの「薄いコンテンツ」ペナルティを避けるため、各ページに固有の解説文（200-300字）を入れる。完全な自動生成テキストのみにしない。

## 3. コンバージョン最大化のサイト設計原則

### 3-1. 記事→CTA→成約の導線設計

**現状の問題**: 記事はあるが、CTAの配置密度と訴求力が弱い可能性がある。

**改善設計**:

```
[記事構成テンプレート]

冒頭リード文（100-200字）
  └ 結論CTA（「結論から言うと〇〇がおすすめ」+ ボタン）... ファーストビュー内

H2: そもそも〇〇とは
  └ 解説テキスト

H2: 〇〇の比較表
  └ 比較表（各行にアフィリリンクボタン）
  └ CTA中間（「迷ったらまずDMM FXの無料口座開設から」+ ボタン）

H2: 選び方のポイント
  └ 解説テキスト

H2: 口座開設の手順（3ステップ図解）
  └ 手順解説
  └ CTA末尾（「最短5分で口座開設完了」+ ボタン）

H2: よくある質問（FAQ）
  └ 信頼醸成・不安解消

末尾CTA（最終クロージング）
```

**CTAの配置ルール**:
1. 記事内に最低3箇所のCTAを配置（冒頭・中間・末尾）
2. 比較表の各行に「公式サイトを見る」リンクを設置
3. ボタン型CTAを使用（バナー広告は広告ブロッカーに弾かれやすいため避ける）
4. CTAテキストは「無料で口座開設」「公式サイトで詳細を確認」等、行動を明示

### 3-2. 比較表・ランキングの効果的な配置

**比較表の設計原則**:
- 横軸: 比較項目（スプレッド、最小単位、キャンペーン、ツール）
- 縦軸: FX会社（最大5社まで。多すぎると選べなくなる）
- 各社のセルに「公式サイト」ボタンを配置
- 数値は出典（各社公式サイト）を明記し、「※2026年6月時点」と日付を入れる
- 「編集部のコメント」欄を1行追加し、差別化ポイントを端的に記載

**ランキングの注意**:
- 景表法・PR表記ルールに準拠。「PR」「広告」の表記を記事冒頭に入れる
- 「1位」等の順位づけの根拠（スプレッド最狭、ユーザー数最多等）を明記
- 「当サイト独自の基準で評価」と書き、客観的指標に基づくことを示す

### 3-3. 心理的トリガー（煽り・虚偽なし）

| トリガー | 実装方法 | 例 |
|---------|---------|-----|
| 損失回避 | 比較表で「知らないと損する」差額を可視化 | 「スプレッド0.1銭の差は10万通貨取引で100円/回。月20回取引すると年間24,000円の差」 |
| 社会的証明 | 口座開設数・取引量の公式データ引用 | 「DMM FXは口座数90万超（2024年時点、公式発表）」 |
| 簡便性 | 口座開設の手順を3ステップで図解 | 「スマホで最短5分。本人確認もオンラインで完結」 |
| 機会提示 | キャンペーン情報を適時更新 | 「2026年7月末まで新規口座開設+取引でキャッシュバック」 |
| 信頼性 | 出典明記+海外FX非推奨の姿勢 | 「当サイトは金融庁登録の国内業者のみを掲載」 |

## 4. 案件別の集中戦略

### 4-1. 最重点案件: DMM FX（月間目標20件 = 44万円）

**理由**:
- 単価27,500円と高額
- 成約条件が「口座開設+入金」のみ（取引条件なしの場合）で敷居が低い
- ブランド認知度が高く、初心者が選びやすい

**攻略記事**:
- `kouza-ranking.html` — ランキング1位にDMM FXを配置（根拠付き）

**理由**:
- 単価53,900円は全案件中最高額

- 競合が少ない（VPS x FXの掛け合わせはニッチ）

**攻略記事**:

### 4-3. ストック収益: TradingView（累積目標100人 = 月6万円）

**理由**:
- ライフタイムコミッション（月額の30%が永続）

- TradingViewのPremiumプランは月$59.95。30% = 約$18 = 約2,700円/月/人
- 100人累積で月27万円（長期的には最大の収益源になりうる）

**攻略記事**:
- `tradingview.html` `tradingview-plan.html` — 既存記事が直接のCV導線
- 新規作成: 「TradingView 無料vs有料プラン比較｜どのプランを選ぶべきか」
- 新規作成: 「TradingView インジケーター おすすめ10選」

**X投稿ネタ**:
- TradingViewのチャート画面キャプチャを使った相場解説（毎日の相場コメントに使用）
- 「TradingViewの便利機能」シリーズ

### 4-4. FTMO（プロップファーム直アフィリ）

**理由**:
- プロップファーム市場は成長中
- 既存記事（ftmo.html, challenge-rules.html等）が5ページあり基盤がある

**注意**: FTMO自体の報酬体系は公式確認が必要。チャレンジ費用の15-20%が相場（仮定）。
